# وحدة التحليل المالي المتقدم والتقارير التنفيذية (Advanced Financial Analysis Module)
## ApexSaaS ERP Engine Documentation

---

### 1. نظرة عامة (Overview)
وحدة **"التحليل المالي المتقدم"** هي وحدة متكاملة للتقرير المالي المتقدم والتحليل الاستراتيجي تم تطويرها لتعمل داخل نظام **ApexSaaS ERP**.
تعتمد الوحدة على خادم خلفي قائم على Node.js / Express مع Drizzle ORM وقواعد بيانات PostgreSQL، وواجهة تفاعلية مبنية بواسطة React 19 + TypeScript + Tailwind CSS + Recharts.

---

### 2. الدقة الحسابية الرياضية (Mathematical Precision Guarantees)
- **الالتزام بشرط عدم استخدام الأرقام العائمة (Decimal Precision Rule):**
  تم إلغاء استخدام `Number()` و `parseFloat()` نهائياً في العمليات المالية الحسابية داخل الـ Repository.
- **مكتبة Decimal.js:**
  يتم استخدام مكتبة `decimal.js` في كافة عمليات الجمع، الطرح، الضرب، القسمة، النسب، وتفكيك المعايير المالية لضمان سلامة العمليات وعدم حدوث أي أخطاء تقريب عائمة (Floating-point precision errors).

---

### 3. نقاط نهاية الخادم الخلفي (Backend API Endpoints)

تم تطبيق حماية الأمان والربط بدليل الصلاحيات لجميع الـ Endpoints عبر middleware `requireScope("accounting:read")` ومرشحات مستأجر النظام `tenantId`:

1. **`GET /api/v1/financial-analysis/trial-balance`**
   - **الوصف:** استخراج ميزان المراجعة الشامل بالأرصدة والمجاميع، والتحقق الحسابي من اتزان إجمالي حركات الدائن والمدين.
2. **`GET /api/v1/financial-analysis/income-statement`**
   - **الوصف:** قائمة الدخل المعيارية وفقاً لـ **IFRS 18** (الفئات الخمس: التشغيلية، الاستثمارية، التمويلية، الضريبية، والعمليات المتوقفة) مع خاصية المقارنة بالسنة السابقة وتحديد التحليل الرأسي الموحد (Common-Size).
3. **`GET /api/v1/financial-analysis/balance-sheet`**
   - **الوصف:** قائمة المركز المالي المقسمة طبقاً لمعيار **IAS 1** إلى أصول والتزامات متداولة وغير متداولة وحقوق ملكية.
4. **`GET /api/v1/financial-analysis/cash-flow`**
   - **الوصف:** قائمة التدفقات النقدية باستخدام الطريقة غير المباشرة (Indirect Method) لمعالجة التسويات النقدية من صافي الربح والأنشطة المختلفة.
5. **`GET /api/v1/financial-analysis/ratios`**
   - **الوصف:** حساب 16+ نسبة مالية متقدمة في السيولة، الربحية، الكفاءة التشغيلية، والرافعة المالية مع مقارنتها بالمعايير المستهدفة (Benchmarks).
6. **`GET /api/v1/financial-analysis/dupont`**
   - **الوصف:** نموذج ديبونت الثلاثي والخماسي الموسّع لتفكيك العائد على حقوق الملكية (ROE) إلى مكوناته الأساسية.
7. **`GET /api/v1/financial-analysis/vertical-analysis`**
   - **الوصف:** التحليل الرأسي المقارن لقائمة الدخل والميزانية العمومية كنسب مئوية من إجمالي الإيرادات أو إجمالي الأصول.
8. **`GET /api/v1/financial-analysis/horizontal-analysis`**
   - **الوصف:** التحليل الأفقي والاتجاهي لمعدلات النمو والتغير عبر الفترات الزمنية (شهري، ربع سنوي، سنوي).
9. **`GET /api/v1/financial-analysis/break-even`**
   - **الوصف:** تحليل نقطة التعادل، هامش المساهمة، ونسبة الأمان المالي.
10. **`GET /api/v1/financial-analysis/altman-z-score`**
    - **الوصف:** نموذج ألت مان (Altman Z-Score) للشركات غير المدرجة للتنبؤ بمخاطر التعثر المالي والإفلاس.
11. **`GET /api/v1/financial-analysis/aging-report`**
    - **الوصف:** تقرير أعمار الديون المتقادمة لذمم العملاء والموردين على فترات استحقاق (0-30، 31-60، 61-90، +90 يوماً).
12. **`GET /api/v1/financial-analysis/cost-center-profitability`**
    - **الوصف:** تقرير أداء ومصروفات مراكز التكلفة ومراكز الربحية مقارنة بالميزانيات والمستهدفات المعتمدة.
13. **`POST /api/v1/financial-analysis/what-if`**
    - **الوصف:** محاكي سيناريوهات الحساسية واختبار افتراضات تغير المبيعات والتكاليف وضريبة الدخل.

### 4. تبويبات واجهة المستخدم (Frontend UI Tabs)

1. **الملخص التنفيذي (Executive Summary):** بطاقات المؤشرات القياسية، الرسم البياني لتوزيع الأرباح، والرادار التفاعلي للسلامة المالية.
2. **القوائم المالية (Financial Statements IFRS 18):** عرض قائمة الدخل المعيارية، ميزانية المركز المالي، التدفقات النقدية، وميزان المراجعة.
3. **النسب المالية (Financial Ratios):** عرض 16+ نسبة مقسمة لفئات مع الشارات الملونة للتقييم.
4. **التحليل الأفقي والرأسي (Trend & Horizontal Analysis):** رسم بياني وجدول لنسب النمو الحجمية والتغير الرأسي.
5. **تحليل ديبونت (DuPont Framework):** الشجرة التفاعلية للنموذج الثلاثي والخماسي الموسّع.
6. **أعمار الديون (Debt Aging Analysis):** تصنيف متقدم للذمم المفتوحة للعملاء والموردين.
7. **ربحية المراكز (Cost & Profit Centers):** متابعة انحرافات الميزانيات لمراكز التكلفة والربحية.
8. **محاكي السيناريوهات (What-If Simulator):** منزلقات تفاعلية لقياس أثر التغيرات المستقبيلية فوراً.
9. **مؤشر Z-Score للإفلاس (Altman Z-Score Risk Gauge):** مقياس سلامة ومخاطر التعثر المالي.
